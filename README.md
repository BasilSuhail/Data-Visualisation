# Data Visualisation

University coursework and projects for Data Visualisation (PX5019), covering both Wolfram Mathematica and Python approaches to visualising data.

## Contents

### Lecture Notebooks (Wolfram Mathematica)
Lectures 1 through 9 covering core data visualisation techniques, from basic plots to advanced statistical graphics, all implemented in Mathematica.

### Exams
| Exam | Tools | Topics |
|---|---|---|
| Exam 1 | Wolfram Mathematica | Applied visualisation problems |
| Exam 2 | Wolfram + Python | Inequality plotting, death causes visualisation |

### Problem Sets
Practice problems with worked solutions in Mathematica.

## Prerequisites

**Wolfram Mathematica** for `.nb` notebook files. A free alternative is Wolfram Player for viewing only.

**Python 3** with `matplotlib` and `pandas` for the Python visualisation scripts:
```bash
pip install matplotlib pandas
```
